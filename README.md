# Service samples

Three small, standalone projects live in the directories below (clone path may match the repository name).

- **`code-smells-1`** — Catalog API (FastAPI + Beanie + MongoDB)
- **`code-smells-2`** — Ops console (Nuxt 3 + TypeScript)
- **`code-smells-3`** — Chat API (FastAPI + Beanie + MongoDB)

Each folder has its own README with how to run it and what it does.
